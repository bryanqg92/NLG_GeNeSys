# Generador de Lenguaje para proceso de enseñanza en una sociedad de robots

## Contexto general 

En este proyecto se ha utilizado los conceptos de lógica difusa y DeepLearning para poder generar texto en forma de instrucciones
teniendo como origen datos númericos decimales.

En la sociedad de robots de GeNeSyS proyecto del MSc. Ing. Dante Sterpin, Se realiza Simulación computarizada del concepto de Neuro-memética, 
donde se heredan comportamientos culturales bajo agentes neuronales llamados Neuro-Nemes. El comporamiento de Wall Following se transmite por varias 
generaciones de robots que utilizan un ANFIS y Un SOM para transmitir los comportamientos de forma numérica bajo 4 variables sensomotrices. P, D, L, y R.

Con el ánimo de querer agregar naturalidad a ese proceso de transmisión y enseñanza cultural, de desarrolla en este repositorio a manera de NOTEBOOKS y source files,
un modelo que es capaz de entender la relación númerica con las instrucciones que se quieren dar en lenguaje natural y que son implementados en el SOFTWARE WEBOTS.
